# HTML Advanced

Projet d'introduction a la structuration de pages web avec HTML5,
realise pour la societe fictive Techium.

## Objectifs

* Creer le squelette d'une page HTML5
* Utiliser les balises semantiques
* Structurer le contenu (titres, listes, tableaux, medias)

## Environnement

* Code conforme W3C
